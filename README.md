Project Overview

Developed a high-performance cross-platform Virtual Tour Platform using Flutter, Dart, Three.js, and WebGL, delivering immersive 3D campus experiences and interactive 360° panorama tours across web and mobile. The application features a custom Flutter ↔ JavaScript bridge enabling seamless communication with embedded WebGL environments for real-time rendering, scene management, and dynamic interactions.

Architected the application with Provider for scalable state management, centralized routing, responsive UI, adaptive themes, multilingual support, persistent user preferences, and enterprise-grade code organization. Built a robust navigation system with custom transitions, error boundaries, and modular architecture for easy scalability.

Implemented advanced Three.js features including GLTF models, OrbitControls, HTML platform views, iframe messaging, and JavaScript interoperability to create smooth, interactive 3D environments. Optimized rendering through adaptive quality scaling, device capability detection, lazy loading, memory management, and efficient asset handling for consistent performance across devices.

Additional features include Firebase Analytics architecture, interactive hotspots, local persistence, comprehensive error handling, reusable components, and a clean, maintainable codebase.

Tech Stack: Flutter, Dart, Provider, Three.js, WebGL, JavaScript, HTML Platform Views, Panorama, Firebase Analytics, Shared Preferences, Responsive UI, Material Design.
